    安裝 golang 在mac 上

    官網安裝步驟

    1.下載安裝包

    安裝來源 golang 1.8.1
    mac 可直接下載 pkg,透過介面安裝

    2. $GOPATH

    $GOPATH 可以設置你的worksapce 存放位置。

    3.設定 Golang 環境變數 for zsh

    ps. go的預設安裝位置可能會因為安裝的方式不同,路徑也有可能不同,建議可以裝完後透過 where go 查看實際路徑

    4. go env

    透過 go env 的命令 可以查看目前 go 相關設置的環境變數
